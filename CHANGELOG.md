# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-01-XX

### Added
- Initial release of Telegram Flick Bot
- Photo upload and processing
- GPT-Image integration for romantic scene creation
- Replicate Wan-Video integration for video generation
- Telegram Stars payment system
- User session management
- Database with Prisma ORM
- MinIO file storage
- Docker support with docker-compose
- Comprehensive documentation
- Error handling and logging
- Graceful shutdown handling

### Features
- Upload 2 photos to create romantic video
- 1 free generation per user
- Buy additional generations with Telegram Stars
- Automatic session management
- Cancel operation support
- Balance checking
- Detailed help system

### Commands
- `/start` - Start the bot
- `/help` - Show help information
- `/generate` - Create romantic video
- `/balance` - Check generation balance
- `/buy` - Purchase generations
- `/cancel` - Cancel current operation
